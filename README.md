# 📸 K-PHOTO - Digital Photocard Gallery (Beta v2.0)

Website galeri interaktif untuk koleksi photocard K-Pop dengan fitur 3D flip card, filter agensi otomatis, dan sistem favorit berbasis LocalStorage.

## ✨ Fitur Utama
- **3D Flip Animation**: Rasakan sensasi membalik photocard asli.
- **Hero Section**: Animasi teks dinamis.
- **Smart Filter**: Filter Big 4 (SM, YG, JYP, HYBE) & kategori 'Other'.
- **Search System**: Pencarian bias secara real-time.
- **Koleksi Saya**: Simpan photocard favorit kamu di browser.

## 🛠️ Teknologi
- HTML5, CSS3 (Flexbox & Grid)
- JavaScript (Vanilla JS - AJAX/Fetch API)
- LocalStorage API
